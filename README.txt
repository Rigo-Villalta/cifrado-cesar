La aplicación consta de 5 módulos (scrips):
-Main
-Configuracion
-Cifrar
-Descifrar
-Créditos

Por defecto el parametro para César es 3

Si no se encuentra el archivo de contraseña, se le solicita al usuario una para almacenarse en formato SHA256

Si no se encuentra el archivo de configuración, se crea con su valor por defecto  de cifrado césar (3)
